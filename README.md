# Hadoop MapReduce WordCount Project

## Description

This project implements a basic WordCount application using Hadoop MapReduce.  
It reads a text file from HDFS, splits the text into words, and calculates the frequency of each word.

## Technologies Used

- Java
- Apache Hadoop
- HDFS
- MapReduce
- YARN
